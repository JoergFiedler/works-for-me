editor
Sublime Text 3 https://www.sublimetext.com/3
Really like it for it's speed and builtin projects-management.
Couple of useful plugins installed:

* Vintageous - vim-mode is must have for me (http://guillermooo.bitbucket.org/Vintageous/)
* AutoFileName (https://github.com/BoundInCode/AutoFileName)
* FileRename (https://github.com/brianlow/FileRename)
* Path Tools (https://github.com/pjdietz/sublime-path-tools)
* Statusbar Path (https://github.com/unphased/SublimeStatusbarPath)

Theme is "Boxy Ocean" (https://github.com/oivva/st-boxy).
Font is awesome "Source Code Pro" from Adobe (https://github.com/adobe-fonts/source-code-pro).
task tracking
Todoist https://todoist.com
Not really fond of this app... Using it like a straight forward checkboxes list.
notes
Notes
Default macOS Notes app has out of the box everything: create notes quickly, simple reach-editing, sync across all devices.
terminal
iTerm 2
Has two features that stop me from switching to default macOS Terminal: * Select to copy - really handy to copy anything from terminal by selecting it * Cmd+click on a link to open it
passwords manager
1Password https://1password.com/
Stores for me not only password but also important documents, notes, credit card, etc.
